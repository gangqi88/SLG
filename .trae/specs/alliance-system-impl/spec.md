## 2. 变更内容

### 2.1 新增功能模块

- **联盟管理模块**: 联盟创建、加入、离开、解散、升级

- **交易系统**: 成员间资源/英雄互换
- **广告系统**: 联盟广告牌、出价竞拍
- **科技系统**: 5种联盟科技升级、属性加成
- **联盟战系统**: 宣战、战斗、结算、奖励发放
- **Web3集成**: 链上NFT、智能合约交互、国库管理

## 4. 需求详情

### 4.1 联盟管理

#### 4.1.1 创建联盟

**Requirement**: 系统应支持玩家消耗 10,000 铜币创建联盟

**Scenario: 成功创建联盟**

- **WHEN** 玩家满足创建条件（无联盟、足够铜币）
- **THEN** 扣除铜币，创建联盟NFT，设置创建者为盟主，返回联盟对象

**Scenario: 创建失败**

- **WHEN** 玩家已有联盟或铜币不足
- **THEN** 返回错误信息，不创建联盟

#### 4.1.2 加入联盟

**Requirement**: 玩家可申请加入联盟，盟主/官员审批后加入

**Scenario: 申请加入**

- **WHEN** 联盟人数未满、玩家无联盟
- **THEN** 创建申请记录，返回等待审批状态

#### 4.1.3 离开联盟

**Requirement**: 玩家可离开联盟，提前24小时申请，离盟扣除50%贡献点

#### 4.1.4 联盟升级

**Requirement**: 联盟可消耗铜币升级，提升成员上限和解锁功能

### 4.2 签到系统

**Requirement**: 每日签到可获得50贡献点，连续签到有额外奖励

**Scenario: 签到成功**

- **WHEN** 玩家今日未签到
- **THEN** 增加贡献点，记录签到时间，返回签到奖励

### 4.3 聊天系统

**Requirement**: 联盟成员可在议事厅发送即时消息

**Scenario: 发送消息**

- **WHEN** 联盟成员发送聊天内容
- **THEN** 创建消息记录，广播给所有在线成员

### 4.4 商店系统

**Requirement**: 联盟成员可使用贡献点购买物品

**Scenario: 购买物品**

- **WHEN** 物品有库存、玩家贡献点足够
- **THEN** 扣除贡献点，减少库存，发放物品

### 4.5 交易系统

**Requirement**: 联盟成员可发起资源/英雄交易请求

### 4.6 广告系统

**Requirement**: 联盟可竞拍主城广告位，每周结算

### 4.7 科技系统

**Requirement**: 联盟可升级5种科技，获得属性加成

| 科技名称 | 效果            | 满级 | 每级成本 |
| -------- | --------------- | ---- | -------- |
| 资源增产 | 全资源+5%/级    | 5级  | 500      |
| 训练加速 | 征兵速度+8%/级  | 5级  | 800      |
| 防御强化 | 城防+10%/级     | 5级  | 1000     |
| 攻击强化 | 野战伤害+8%/级  | 5级  | 1000     |
| 采集增益 | 采集速度+15%/级 | 3级  | 1500     |

### 4.8 联盟战

**Requirement**: 联盟等级≥5可宣战，24小时战斗时间

**Scenario: 宣战**

- **WHEN** 联盟等级≥5、拥有宣战令、保证金充足
- **THEN** 锁定保证金，创建战争记录

### 4.9 Web3集成

**Requirement**: 支持链上联盟NFT、国库存取、事件监听
