# 《指尖无双》手游 · 界面整合验收清单（V1.0）

## A. 全局规范

- [ ] 全局主背景/面板/边框色值对齐：`#1A1A2A / #25253A / #3A3A5A`
- [ ] 标题金色与正文白色、辅助灰色对齐：`#D4B87E / #FFFFFF / #9999AA`
- [ ] 语义按钮色对齐：升级绿、活动黄、战斗红、信息蓝
- [ ] 品质色对齐：绿/蓝/紫/橙/红（且用于武将/装备/宝箱边框）
- [ ] 字号档位齐全：大标题/模块标题/正文/辅助；数字属性使用等宽字体
- [ ] 圆角统一为 12；列表间距默认 16
- [ ] 所有可点击按钮/入口满足最小 88×88 点击区域
- [ ] 竖屏优先，安全区（刘海/底部手势条）不遮挡顶部/底部关键按钮
- [ ] 720P/1080P/2K 自适应布局不拉伸文字变形

- [ ] 按钮按压动效：缩放 0.95→回弹 1.0 + 颜色变亮
