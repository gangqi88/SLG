# R1 语言与术语规则

## 语言要求

- 所有技能文档使用中文（简体）
- 代码块内可使用英文标识
- 技术术语首次出现需给出中文解释

## 标点与表达

- 正文使用中文标点
- 列表项优先短句，避免冗长
- 数字统一使用阿拉伯数字

## 术语统一

| 术语     | 约束             |
| -------- | ---------------- |
| 代码审查 | 不写 code review |
| 持续集成 | 不写 CI          |
| 持续交付 | 不写 CD          |
| 单元测试 | 不写 unit test   |

## 示例

- 正确：单元测试覆盖率
- 不推荐：unit test coverage
