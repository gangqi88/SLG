# R4 职责边界与协作接口

## 职责边界

- `task-scheduler`：只做任务路由与编排
- `web3-slg-project-manager`：只做规划与跟踪
- `fullstack-code-standards`：只做开发落地与代码规范
- `game-tester`：只做测试设计、执行与验证
- `document-reviewer`：只做文档审查与一致性检查

## 协作接口

输入建议：

- 任务目标
- 优先级
- 依赖关系

输出建议：

- 处理结果
- 风险项
- 下一技能交接说明

## 去重原则

- 全局规则不在技能文档重复粘贴
- 技能文档只保留本技能独有内容
