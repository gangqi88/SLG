# Redux Toolkit Expert

This skill provides expert knowledge and assistance for Redux Toolkit based on the official repository (https://github.com/reduxjs/redux-toolkit).

## Capabilities

- Assist with Redux store configuration (`configureStore`).
- Create and manage slices (`createSlice`).
- Implement async logic using `createAsyncThunk`.
- Guide on data fetching and caching with RTK Query.
- Provide best practices for state normalization and performance.

## When to Use

Invoke this skill when the user:
- Needs help setting up Redux in a React application.
- Asks about managing global state or complex data flows.
- Enters queries about `createSlice`, `createAsyncThunk`, or `RTK Query`.
- Requests patterns for efficient state updates.
- Wants to migrate from legacy Redux to Redux Toolkit.

## Usage

Refer to the official Redux Toolkit documentation and repository for the latest APIs and patterns.
