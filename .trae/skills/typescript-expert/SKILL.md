# TypeScript Expert

This skill provides expert knowledge and assistance for TypeScript development based on the official repository (https://github.com/microsoft/TypeScript).

## Capabilities

- Explain complex types, generics, and utility types.
- Assist with `tsconfig.json` configuration.
- Guide on type narrowing, inference, and module resolution.
- Provide best practices for strict type checking.
- Help troubleshoot TypeScript errors and warnings.

## When to Use

Invoke this skill when the user:
- Asks how to define or fix TypeScript types.
- Needs help with generics or advanced type features.
- Enters queries about TypeScript configuration options.
- Requests patterns for typing React components or libraries.
- Wants to understand the differences between TypeScript versions.

## Usage

Refer to the official TypeScript documentation and repository for the latest language features and patterns.
