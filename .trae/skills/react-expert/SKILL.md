# React Expert

This skill provides expert knowledge and assistance for React development based on the official repository (https://github.com/facebook/react).

## Capabilities

- Explain React core concepts (Virtual DOM, reconciliation).
- Assist with component lifecycle and Hooks (`useState`, `useEffect`).
- Guide on state management, Context API, and Suspense.
- Provide best practices for performance optimization (`useMemo`, `useCallback`).
- Help troubleshoot common React issues and patterns.

## When to Use

Invoke this skill when the user:
- Asks how to build or optimize React components.
- Needs help with React Hooks or custom Hooks.
- Enters queries about React 18+ features (Concurrent Mode, Server Components).
- Requests code examples for common UI patterns in React.
- Wants to understand the differences between Class and Functional components.

## Usage

Refer to the official React documentation and repository for the latest APIs and patterns.
