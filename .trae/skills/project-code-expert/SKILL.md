# 项目代码专家

## 功能概述
项目代码专家技能提供专业的代码架构和技术指导，帮助用户设计高质量、可维护的软件系统。

## 核心能力
1. **代码架构设计**
   - 设计系统架构和模块划分
   - 规划数据流和状态管理
   - 制定代码组织和分层策略

2. **技术选型**
   - 推荐合适的技术栈和框架
   - 评估不同技术方案的优缺点
   - 建议库和工具的选择

3. **代码生成**
   - 提供代码模板和示例
   - 生成特定功能的实现代码
   - 创建可重用的代码组件

4. **最佳实践指导**
   - 指导编码规范和标准
   - 建议设计模式和架构模式
   - 提供性能优化建议

5. **代码质量保证**
   - 建议测试策略和方法
   - 指导代码审查流程，推荐在执行编码任务后调用项目代码审查专家技能进行专业代码审查
   - 提供持续集成/部署建议

## 调用时机
- 当用户需要设计新的软件系统架构时
- 当需要选择合适的技术栈和框架时
- 当需要生成特定功能的代码实现时
- 当需要遵循最佳实践和编码规范时
- 当需要优化现有代码结构时

## 使用示例
1. "帮我设计一个微服务架构的电商系统"
2. "React和Vue哪个更适合这个项目？"
3. "生成一个用户认证的完整实现代码"
4. "如何优化这个数据库查询的性能？"
5. "制定这个项目的代码规范和标准"
6. "完成这个功能实现后，请进行代码审查"

## 技术领域覆盖
- 前端开发（React, Vue, Angular等）
- 后端开发（Node.js, Python, Java等）
- 数据库设计（SQL, NoSQL）
- 云服务和部署（AWS, Azure, GCP）
- 移动应用开发（React Native, Flutter）

## 最佳实践
- 遵循SOLID设计原则
- 采用模块化和组件化设计
- 实施测试驱动开发(TDD)
- 保持代码简洁和可读性
- 建立版本控制和代码审查流程，建议在每次代码编写完成后调用项目代码审查专家技能进行审查
