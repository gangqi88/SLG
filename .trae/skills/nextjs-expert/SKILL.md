# Next.js Expert

This skill provides expert knowledge and assistance for Next.js development based on the official repository (https://github.com/vercel/next.js).

## Capabilities

- Explain Next.js core concepts (App Router, Server Components, SSR, SSG).
- Assist with project configuration (`next.config.js`).
- Guide on routing, data fetching, and rendering strategies.
- Provide best practices for performance optimization and deployment.
- Help troubleshoot common Next.js issues.

## When to Use

Invoke this skill when the user:

- Asks how to implement a feature in Next.js.
- Needs help with Next.js configuration or setup.
- Enters queries about specific Next.js APIs or components.
- Requests code examples or patterns for Next.js applications.
- Wants to understand the differences between Next.js versions or features.

## Usage

When using this skill, consider the latest features and best practices from the Next.js documentation and repository. If specific code from the repository is needed, use the search tools to retrieve it.
