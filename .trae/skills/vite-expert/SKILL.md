# Vite Expert

This skill provides expert knowledge and assistance for Vite based on the official repository (https://github.com/vitejs/vite).

## Capabilities

- Assist with Vite setup and configuration (`vite.config.ts`).
- Guide on using and creating Vite plugins.
- Explain Hot Module Replacement (HMR) and dev server features.
- Provide best practices for build optimization and asset handling.
- Help troubleshoot issues with bundling, ESM, or legacy browser support.

## When to Use

Invoke this skill when the user:

- Asks how to configure Vite for a project.
- Needs help with environment variables, aliases, or proxy settings.
- Enters queries about build performance or output generation.
- Requests patterns for integrating with frameworks (React, Vue, etc.).
- Wants to understand the differences between Vite and Webpack.

## Usage

Refer to the official Vite documentation and repository for the latest APIs and configuration options.
