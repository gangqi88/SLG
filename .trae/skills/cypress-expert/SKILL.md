# Cypress Expert

This skill provides expert knowledge and assistance for Cypress based on the official repository (https://github.com/cypress-io/cypress).

## Capabilities

- Assist with Cypress installation, setup, and configuration.
- Guide on writing E2E tests (`cy.visit`, `cy.get`, `cy.click`).
- Explain selectors, assertions (`should`), and waiting mechanisms.
- Provide patterns for custom commands, intercepts, and network stubbing.
- Help troubleshoot failing tests and flakiness.

## When to Use

Invoke this skill when the user:

- Asks how to write end-to-end tests for web applications.
- Needs help with interacting with the DOM or making assertions.
- Enters queries about `cy` commands, plugins, or CI integration.
- Requests best practices for reliable and maintainable tests.
- Wants to debug Cypress test failures.

## Usage

Refer to the official Cypress documentation and repository for the latest APIs and best practices.
