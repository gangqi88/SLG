# 项目管理专家

## 功能概述
项目管理专家技能提供专业的项目管理指导和支持，帮助用户规划、执行和监控软件项目。

## 核心能力
1. **项目规划**
   - 制定项目时间表和里程碑
   - 定义项目范围和需求
   - 创建项目文档和路线图

2. **进度跟踪**
   - 监控项目进展状态
   - 识别进度偏差和延迟
   - 提供进度报告模板

3. **资源分配**
   - 优化团队资源配置
   - 平衡工作负载分配
   - 建议资源调配策略

4. **风险评估**
   - 识别项目潜在风险
   - 制定风险缓解计划
   - 提供应急方案建议

5. **沟通管理**
   - 制定项目沟通计划
   - 建议会议和报告频率
   - 提供利益相关者管理指导

## 调用时机
- 当用户需要制定新的项目计划时
- 当项目遇到进度延迟或资源问题时
- 当需要评估项目风险和制定应对策略时
- 当需要优化项目管理流程时
- 当需要制定项目文档和报告模板时

## 使用示例
1. "帮我制定一个React项目的开发计划"
2. "我的项目进度落后了，有什么建议？"
3. "如何分配团队资源来提高效率？"
4. "识别这个项目的潜在风险有哪些？"
5. "制定一个软件项目的沟通计划"

## 最佳实践
- 使用SMART原则设定项目目标
- 定期进行项目健康检查
- 建立透明的沟通机制
- 采用敏捷或瀑布方法学
- 持续跟踪关键绩效指标(KPI)
