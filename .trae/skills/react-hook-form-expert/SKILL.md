# React Hook Form Expert

This skill provides expert knowledge and assistance for React Hook Form based on the official repository (https://github.com/react-hook-form/react-hook-form).

## Capabilities

- Assist with form setup (`useForm`, `useController`).
- Implement form validation (native, Yup, Zod, etc.).
- Guide on handling controlled vs uncontrolled components.
- Explain performance optimization and re-render management.
- Provide patterns for complex forms, arrays (`useFieldArray`), and nested fields.

## When to Use

Invoke this skill when the user:

- Asks how to build or manage forms in React.
- Needs help with validation rules, error handling, or submission.
- Enters queries about `register`, `handleSubmit`, or `Controller`.
- Requests patterns for integrating with UI libraries (MUI, Chakra, etc.).
- Wants to optimize form performance and reduce re-renders.

## Usage

Refer to the official React Hook Form documentation and repository for the latest APIs (v7+) and patterns.
