# React Router Expert

This skill provides expert knowledge and assistance for React Router based on the official repository (https://github.com/remix-run/react-router).

## Capabilities

- Assist with router setup (`createBrowserRouter`, `RouterProvider`).
- Define routes and nested routes.
- Implement data loading and mutations (`loader`, `action`).
- Guide on navigation (`Link`, `useNavigate`).
- Provide best practices for routing in React apps.

## When to Use

Invoke this skill when the user:
- Asks how to set up routing in a React application.
- Needs help with route parameters, nested routes, or layouts.
- Enters queries about React Router 6+ features like data routers.
- Requests patterns for protected routes or authentication flows.
- Wants to understand the differences between React Router versions.

## Usage

Refer to the official React Router documentation and repository for the latest APIs and patterns.
