# 项目代码审查专家

## 功能概述
项目代码审查专家技能提供专业的代码审查和分析，集成ESLint和SonarQube静态代码分析工具，帮助用户提高代码质量、安全性和可维护性，确保遵循最佳实践并通过自动化代码检查。

## 核心能力
1. **代码质量检查**
   - 检查代码可读性和可维护性
   - 评估代码复杂度和耦合度
   - 识别代码异味和反模式

2. **安全漏洞检测**
   - 识别常见安全漏洞（SQL注入、XSS等）
   - 检查敏感数据泄露风险
   - 验证输入验证和输出编码

3. **性能问题识别**
   - 发现性能瓶颈和低效代码
   - 检查内存泄漏和资源管理
   - 评估算法和数据结构效率

4. **最佳实践审核**
   - 验证编码规范遵循情况
   - 检查设计模式应用正确性
   - 评估错误处理和日志记录

5. **架构一致性检查**
   - 确保代码符合架构设计
   - 检查模块边界和接口定义
   - 验证依赖管理和包结构

6. **静态代码分析工具集成**
   - 集成ESLint进行JavaScript/TypeScript代码规范检查
   - 配置和运行SonarQube进行全面的代码质量分析
   - 检查ESLint规则遵循情况和SonarQube质量问题
   - 提供ESLint和SonarQube检查结果解读和改进建议

## 调用时机
- 当用户需要专业的代码质量评估时
- 当需要检查代码安全漏洞和风险时
- 当需要识别和修复性能问题时
- 当需要确保代码遵循最佳实践时
- 当需要审查代码架构一致性时
- 当需要通过ESLint和SonarQube进行自动化代码检查时

## 使用示例
1. "请审查这段React组件的代码质量"
2. "检查这段代码是否存在安全漏洞"
3. "识别这段数据库查询的性能问题"
4. "评估这个API服务的错误处理是否充分"
5. "审查这个模块是否符合项目架构规范"
6. "运行ESLint和SonarQube检查这段代码并解读结果"

## 审查维度
1. **代码质量维度**
   - 可读性：命名规范、注释清晰度
   - 可维护性：模块化、复杂度控制
   - 可测试性：测试覆盖率、可测试性设计

2. **安全维度**
   - 输入验证和输出编码
   - 认证和授权实现
   - 敏感数据处理和保护

3. **性能维度**
   - 算法时间和空间复杂度
   - 数据库查询优化
   - 内存管理和资源释放

4. **最佳实践维度**
   - 设计模式应用
   - 异常处理策略
   - 日志记录和监控

5. **静态分析工具维度**
   - ESLint规则遵循情况和错误/警告数量
   - SonarQube代码质量门禁通过情况
   - 技术债务和代码覆盖率指标
   - 安全热点和漏洞检测结果

## 审查流程
1. **工具自动化扫描**：使用ESLint和SonarQube进行快速扫描，识别明显问题
2. **深度分析**：详细检查关键代码段，结合工具分析结果进行人工验证
3. **问题分类**：按严重程度和类型分类，区分工具检测问题和人工发现问题
4. **建议提供**：具体改进建议和示例代码，包含ESLint规则修复和SonarQube问题解决方案
5. **优先级排序**：建议修复顺序和紧急程度，考虑工具报告的严重性等级

## 交付物
- 代码审查报告
- 问题清单和严重程度评级
- 具体改进建议和示例
- 最佳实践参考和资源链接
- ESLint和SonarQube检查报告
- 后续行动计划和跟踪建议
