# Chakra UI Expert

This skill provides expert knowledge and assistance for Chakra UI based on the official repository (https://github.com/chakra-ui/chakra-ui).

## Capabilities

- Assist with Chakra UI installation and setup (`ChakraProvider`).
- Use Chakra UI components (Box, Flex, Button, Input, etc.).
- Customize themes, colors, and fonts (`extendTheme`).
- Guide on responsive design and style props.
- Provide best practices for accessible and scalable UI development.

## When to Use

Invoke this skill when the user:

- Asks how to implement UI components with Chakra UI.
- Needs help with theming or styling Chakra UI components.
- Enters queries about responsive layouts or style props.
- Requests patterns for building design systems with Chakra UI.
- Wants to troubleshoot styling issues in Chakra UI.

## Usage

Refer to the official Chakra UI documentation and repository for the latest component APIs and theming options.
