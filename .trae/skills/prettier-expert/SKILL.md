# Prettier Expert

This skill provides expert knowledge and assistance for Prettier based on the official repository (https://github.com/prettier/prettier).

## Capabilities

- Assist with Prettier installation and configuration (`.prettierrc`).
- Guide on formatting rules and options (print width, tabs, etc.).
- Explain how to integrate Prettier with ESLint, editors, and CI.
- Provide best practices for consistent code style across a project.
- Help troubleshoot formatting issues and conflicts.

## When to Use

Invoke this skill when the user:

- Asks how to set up automatic code formatting.
- Needs help with configuring Prettier options.
- Enters queries about code style enforcement or editor integration.
- Requests patterns for avoiding conflicts with linters.
- Wants to understand how Prettier handles different file types.

## Usage

Refer to the official Prettier documentation and repository for the latest options and supported languages.
