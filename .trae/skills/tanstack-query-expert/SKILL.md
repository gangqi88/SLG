# TanStack Query Expert

This skill provides expert knowledge and assistance for TanStack Query (formerly React Query) based on the official repository (https://github.com/TanStack/query).

## Capabilities

- Assist with setup and `QueryClient` configuration.
- Implement data fetching (`useQuery`) and mutations (`useMutation`).
- Guide on query invalidation, refetching, and pagination.
- Explain caching strategies, stale time, and background updates.
- Provide best practices for managing server state and optimistic updates.

## When to Use

Invoke this skill when the user:

- Asks how to fetch data in React (or other frameworks supported by TanStack Query).
- Needs help with caching, loading states, or error handling.
- Enters queries about `useQuery`, `useMutation`, or `useInfiniteQuery`.
- Requests patterns for data synchronization and optimistic UI.
- Wants to optimize data fetching performance.

## Usage

Refer to the official TanStack Query documentation and repository for the latest APIs (v4/v5+) and patterns.
