# Immer Expert

This skill provides expert knowledge and assistance for Immer based on the official repository (https://github.com/immerjs/immer).

## Capabilities

- Assist with using `produce` to update immutable state.
- Explain how proxies work for drafting state.
- Guide on updating nested objects and arrays immutably.
- Provide patterns for integrating with React (`useImmer`) or Redux (often built-in).
- Help troubleshoot issues with state mutations or patches.

## When to Use

Invoke this skill when the user:
- Asks how to update complex nested state in JavaScript/React.
- Needs help with simplifying reducer logic.
- Enters queries about `produce`, drafts, or patches.
- Requests patterns for safe state mutation.
- Wants to understand the benefits of Immer over manual immutability.

## Usage

Refer to the official Immer documentation and repository for the latest APIs and patterns.
