# Emotion Expert

This skill provides expert knowledge and assistance for Emotion (CSS-in-JS library) based on the official repository (https://github.com/emotion-js/emotion).

## Capabilities

- Assist with Emotion setup and configuration (`@emotion/react`, `@emotion/styled`).
- Guide on using the `css` prop and `styled` API.
- Explain composition, theming, and global styles.
- Provide best practices for performance and server-side rendering (SSR).
- Help troubleshoot styling issues and TypeScript integration.

## When to Use

Invoke this skill when the user:
- Asks how to style components using Emotion.
- Needs help with dynamic styling or theming.
- Enters queries about the difference between `css` prop and `styled`.
- Requests patterns for reusable styled components.
- Wants to migrate from other CSS-in-JS libraries to Emotion.

## Usage

Refer to the official Emotion documentation and repository for the latest APIs and patterns.
