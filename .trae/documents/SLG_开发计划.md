# 三国·神魔人族 SLG 游戏开发计划

> 基于《Web3游戏项目-SLG.md》策划文档重构

## 项目概述

本项目是一款基于 **React + TypeScript + Phaser 3** 的三国SLG游戏，包含三族阵营对抗（人族、天使、恶魔）、30名原创英雄养成、策略战斗等核心玩法。

### 第二阶段：技能与战斗系统 ⭐⭐⭐⭐⭐

#### 2.1 技能系统实现

**技能类型**：
- 主动技能（Active）：冷却时间触发
- 被动技能（Passive）：常驻生效
- 天赋技能（Talent）：英雄独特加成

**技能效果分类**：
```typescript
type SkillEffectType = 
  | 'damage'        // 伤害
  | 'heal'          // 治疗
  | 'buff'          // 增益
  | 'debuff'        // 减益
  | 'shield'        // 护盾
  | 'revive'        // 复活
  | 'special';      // 特殊效果
```

**冷却时间标准**：
- 紫将主动：15-20秒
- 橙将主动：12-18秒
- 红将主动：8-15秒
- 复活/无敌类：全场仅限1次

#### 2.2 战斗系统实现

**战斗流程**：
1. 队伍配置（主将+副将×2）
2. 战斗开始（自动/手动）
3. 技能释放（冷却/能量）
4. 伤害计算（属性/克制/Buff）
5. 战斗结束（胜负判定/战报）

**伤害公式**（按策划文档）：
```
普攻伤害 = 武力 × 0.8
技能伤害 = 谋略 × 1.0~1.8
治疗量 = 谋略 × 1.2
减免比例 = 防御 / (防御 + 200)
```

**阵营克制**：
- 恶魔 → 人族 +25%
- 人族 → 天使 +20%
- 天使 → 恶魔 +30%

### 第三阶段：阵容与队伍系统 ⭐⭐⭐⭐

#### 3.1 阵容系统

**队伍配置**：
- 单队：主将1名 + 副将2名
- 最大出征：3队同时

**推荐阵容**（按策划文档）：

**人族·稳守发育流**
- 主将：钟离野（先锋战士）
- 副将：秦烈（坦克）+ 苏晚晴（治疗）
- 内政：苏墨 + 梁石
- 特点：资源爆炸、守城极强

**天使·圣光永动流**
- 主将：米迦尔（审判输出）
- 副将：洛曦（治疗）+ 迦南（防御）
- 辅助：艾琳娜 + 乌列
- 特点：治疗拉满、克制恶魔

**恶魔·炼狱爆杀流**
- 主将：巴尔（野战战神）
- 副将：摩洛克（AOE）+ 卡隆（收割）
- 功能：阿加雷斯 + 玛门
- 特点：爆发秒杀、掠夺攻城
